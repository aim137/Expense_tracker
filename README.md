# Crypto trading bot

This is a python application to track expenses and split costs with another person.
 
## Installation


```bash
clone repo
cd to repo
pip install -e .
```
Make sure to edit xp/defaults.py with your names and default expenses.

## Usage

```python
from expensetracker.driver import run

run()

```
